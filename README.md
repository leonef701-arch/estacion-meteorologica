🛠️ Tecnologías Utilizadas
En este proyecto integramos diversas herramientas distribuidas en tres áreas clave:

🔬 Club de CIENCIAS (Análisis y Datos)
Google Colab - Prototipado de datos y uso de librerías científicas.
MySQL Workbench - Diseño y gestión de la base de datos relacional.
Power BI - Creación de dashboards interactivos para la visualización del clima.
🐍 PROGRAMACION III (Software)
Visual Studio Code - Entorno de desarrollo (IDE) principal.
Python - Lenguaje de programación con enfoque en POO (Programación Orientada a Objetos).
Librerías de Conexión: Uso de conectores para vincular la lógica de Python con la base de datos.
🤖 ROBOTICA (Hardware y Simulación)
Arduino IDE - Programación del microcontrolador en C++.
Tinkercad - Simulación del circuito electrónico y sensores.
Componentes: Arduino UNO R3, sensores de temperatura, humedad y presión para el prototipo físico.
